# Difference Between `elif` and Multiple `if` Statements

## 1. Multiple `if` Statements
- Each `if` condition is evaluated **independently**.
- If multiple conditions are **True**, multiple blocks will execute.

Example:

if condition1:
    # executes if condition1 is True

if condition2:
    # also executes if condition2 is True


if condition1:
    # executes if condition1 is True
elif condition2:
    # executes only if condition1 is False and condition2 is True


Example code:
x = 15

if x > 10:
    print("Greater than 10")

if x > 5:
    print("Greater than 5")

output:
Greater than 10
Greater than 5

x = 15

if x > 10:
    print("Greater than 10")
elif x > 5:
    print("Greater than 5")

output:
Greater than 10
